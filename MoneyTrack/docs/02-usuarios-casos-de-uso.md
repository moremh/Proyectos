# MoneyTrack - Usuarios y Casos de Uso

# 1. Tipos de usuarios

## Usuario

Es el usuario principal de la aplicación.

Puede:

- Registrarse.
- Iniciar sesión.
- Recuperar su contraseña.
- Registrar ingresos.
- Registrar gastos.
- Crear categorías.
- Editar categorías.
- Eliminar categorías.
- Consultar estadísticas.
- Crear objetivos de ahorro.
- Editar su perfil.
- Cerrar sesión.

---

# 2. Casos de uso

## CU-01 - Registrarse

**Actor:** Usuario

**Descripción:**

El usuario crea una cuenta proporcionando sus datos personales para comenzar a utilizar la aplicación.

---

## CU-02 - Iniciar sesión

**Actor:** Usuario

**Descripción:**

El usuario ingresa su correo electrónico y contraseña para acceder a su cuenta.

---

## CU-03 - Registrar un ingreso

**Actor:** Usuario

**Descripción:**

El usuario registra un nuevo ingreso indicando el monto, la categoría, la fecha y una descripción opcional.

---

## CU-04 - Registrar un gasto

**Actor:** Usuario

**Descripción:**

El usuario registra un gasto indicando el monto, la categoría, la fecha y una descripción opcional.

---

## CU-05 - Administrar categorías

**Actor:** Usuario

**Descripción:**

El usuario puede crear, modificar y eliminar categorías para organizar mejor sus movimientos financieros.

---

## CU-06 - Consultar Dashboard

**Actor:** Usuario

**Descripción:**

El usuario visualiza un resumen con:

- Saldo total.
- Ingresos del mes.
- Gastos del mes.
- Balance mensual.
- Últimos movimientos.
- Gráficos estadísticos.

---

## CU-07 - Consultar reportes

**Actor:** Usuario

**Descripción:**

El usuario puede visualizar reportes y gráficos sobre sus finanzas para analizar sus hábitos de gasto e ingreso.

---

## CU-08 - Gestionar objetivos de ahorro

**Actor:** Usuario

**Descripción:**

El usuario crea objetivos de ahorro y realiza un seguimiento de su progreso.

---

## CU-09 - Editar perfil

**Actor:** Usuario

**Descripción:**

El usuario puede modificar su información personal y cambiar su contraseña.

---

## CU-10 - Cerrar sesión

**Actor:** Usuario

**Descripción:**

El usuario finaliza su sesión de forma segura.
