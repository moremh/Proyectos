# MoneyTrack - Diseño

## Objetivo visual

Crear una interfaz moderna, limpia e intuitiva que permita visualizar la información financiera de manera rápida y sencilla.

---

# Estilo visual

Inspiraciones:

- Notion
- Stripe
- Linear
- GitHub

Características:

- Diseño minimalista.
- Espacios amplios.
- Tarjetas con bordes redondeados.
- Iconografía simple.
- Responsive.
